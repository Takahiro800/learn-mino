### Javaのアクセス修飾子
- `public` すべてのクラスからアクセス可能
- `protected` 同じクラス、または継承クラスからアクセス可能。
- なし　同じパッケージからのみアクセス可能。package privateと呼ぶ。
- `private` 同じクラスからのみアクセス可能
